Freak Cloud Alpha Node on zkSync testnet
The Freak Cloud Alpha node has officially launched on the zkSync testnet. This Alpha node may contain bugs, but we hope it provides you with a basic understanding of how Freak Cloud works. Our team is dedicated to regularly updating and refining the system!

Faucet for Freak Cloud Alpha node/client In order to use Freak Cloud Alpha Testnet, users should receive testnet token on zkSync Testnet(Goerli)
1. Receive $FREAKC testnet token
To claim from the Freak Cloud testnet faucet: * Join Freak Cloud Discord and navigate to the #testnet-faucet channel * Once there, type the command /claim [your address]
2. Receive $Geth on zkSync Goerli Testnet for paying gas fees
(If you familiar with how to claim GETH token of zkSync Testnet, you can skip this) Step 1: Claim GETH of Goerli Testnet Step 2: Go to zkSync Goerli Testnet Bridge Step 3: Connect Wallet and transfer GETH token from Goerli Testnet to zkSync Goerli Testnet
