Requirement

Ensure you have sufficient FREAKC tokens and Sepolia ETH tokens on the zkSync Sepolia Testnet to use the node. If not, claim some at #Receive testnet coins for testing.

â˜ï¸ Freak Incentive Model, Min reward, Max rewardâ€‹

Freak Incentive Modelâ€‹

In storage agreement, when users request data, fastest storage providers delivering data receive 1 Freak Point (Just be online; your node/client will handle it automatically).

In each storage agreement(storage contract):

  • 75% value of storage agreement/storage contract distributed to all storage providers
  • 25% value distributed to top 1/3 storage provider have the highest Freak Points (this is Active Rank Bonus Reward)

    Freak Incentive model fosters competition among storage providers, ensuring best service for users.

Min rewardâ€‹

All providers in storage agreement receive this reward.

Max rewardâ€‹

Max reward = Min reward + Active rank bonus reward In storage agreement, fastest providers sending data upon user download requests receive 1 Freak Point.

(Just be online; your node/client will handle it automatically)

In each storage agreement, only the top 1/3 active providers (with the highest Freak Points) receive the Active Rank Bonus Reward.

Conclusion: The more you are online, the higher your chance of receiving the Active Rank Bonus Reward.
