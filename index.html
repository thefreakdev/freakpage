Hyper gas fee minimization
Ensuring that data is properly stored and significantly reducing verification cost and proof sizes. For example, if a direct verification normally needs 1000000 instructions, Merkle proof just needs log2(1000000) = 19.93 instructions.

Why choose Freak Cloud

Innovation

Freak Cloud is the first-ever decentralized data storage being able to run on zkEVM enviroment.

Decentralization

User data is anonymously stored by multiple storage providers, and anyone can become a storage provider. Freak Cloud fosters true decentralization in data storage.

Competitive storage price

Uncensored storage pricing determined by users and providers. An order book system lets users and providers place market or spot price orders for storage, seamlessly matching supply and demand.

Exclusive for zkSync

Users and dapps on zkSync can securely and efficiently interact with Freak Cloud without the need for cross-chain communication and bridge fees.

Efficiency

By being compatible with zk-rollup of zkSync and using Merkle-tree proofs, Freak Cloud is able to achieve high performance and hyper gas fee minimization.

Scability

A combination of zkSync environment, storage cryptographic proof and incentivization model allows Freak Cloud to achieve high and sustainable scaling.

Security and privacy

Data uploaded by users are encrypted. There is no KYC or whatsoever, information of users and storage providers are anounymous.

Reliable

A trustless model systematically eradicates fraudulent activities, solves data witholding problems and ensures providers give out best service for users.

Simplicity

Freak Cloud delivers a seamless, fast, and user-friendly experience through its simple interface for crypto and non-crypto users.
